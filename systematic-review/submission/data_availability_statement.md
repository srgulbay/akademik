# Data Availability Statement

This systematic review used only previously published, peer-reviewed literature. No new patient data and no new individual-level human or animal data were generated. All study-level data extracted from the included records, all search strategies, the PRISMA 2020 checklist and the custom analytic code are openly available as described below.

## Supplementary materials accompanying this article

| File | Content | Format |
|---|---|---|
| **S1** | Completed PRISMA 2020 checklist | .docx / .pdf |
| **S2** | Full search strategies for each database (PubMed/MEDLINE, planned Scopus, Web of Science Core Collection, Embase, Cochrane Library) and the OpenAlex/Crossref API calls used for cross-validation | .docx / .pdf |
| **S3** | PRISMA 2020 flow data (record counts at every identification, screening, eligibility and inclusion stage; cross-database overlap matrix) | .csv / .json |
| **S4** | Standardised data-extraction form (bibliographic identifiers, study design, population, intervention, outcomes, risk-of-bias domains, evidence-claim summary) | .docx / .pdf |
| **S5** | Categorised corpus (340 included records with topic, study-design and intervention-class tags) | .csv |
| **S6** | Evidence-claims database (structured per-study extractions linking quantitative outcomes to source records, with effect sizes, confidence intervals and p-values where reported) | .json |
| **S7** | Full bibliography of included records (n = 340) with DOI, PMID and PMCID | .md / .docx / .json |
| **S8** | Citation shortlist (top records by importance score used in the priority synthesis) | .csv |
| **S9** | Figure source files (vector SVG and 300 DPI PNG for Figures 1–6) and the Python scripts used to generate them | .zip |
