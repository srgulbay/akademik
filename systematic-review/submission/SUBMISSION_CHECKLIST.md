# IJAA Submission Checklist

**Manuscript:** *Quorum Sensing in Acinetobacter baumannii: Molecular Architecture, Therapeutic Targeting and Translational Horizons — A Systematic Review (2003–2025)*
**Target journal:** International Journal of Antimicrobial Agents (IJAA), Elsevier
**Submission portal:** Editorial Manager (https://www.editorialmanager.com/ijaa/)

Use this checklist as a final pre-submission verification. Tick each box only after the underlying artefact has been produced and verified.

## A. Core narrative files

- [ ] Cover letter prepared and personalised (`/submission/cover_letter.docx`); placeholders for author name, affiliation, contact details, suggested reviewers and prior-dissemination/preprint disclosure completed.
- [ ] Title page complete with all required elements (`/submission/title_page.docx`):
  - [ ] Manuscript title (≤ 150 characters, confirmed)
  - [ ] Running head
  - [ ] All authors with superscript affiliation numerals
  - [ ] Numbered affiliations
  - [ ] Corresponding author block (name, address, telephone, email, ORCID)
  - [ ] Word counts (abstract, main text)
  - [ ] Number of tables, figures, references and supplementary files
  - [ ] Keywords (4–7)
  - [ ] Conflicts of interest statement
  - [ ] Funding statement
  - [ ] Ethical approval statement (or N/A justification for SRs)
  - [ ] Data availability statement
  - [ ] CRediT author contributions
  - [ ] Acknowledgements
  - [ ] Prior dissemination / preprint disclosure
- [ ] Main manuscript file (Word, `.docx`) prepared with:
  - [ ] Double-spaced text
  - [ ] Continuous line numbering
  - [ ] All citations rendered in numbered Vancouver style consistent with the reference list
  - [ ] Highlights (3–5 bullets, each ≤ 85 characters including spaces)
  - [ ] Structured abstract (Background / Objective / Methods / Results / Conclusions, ≤ 300 words)
  - [ ] Keywords (4–7)
  - [ ] Main text ≤ 10,000 words (excluding references, tables, figure legends) — or written justification for extended length in the cover letter
  - [ ] Data Availability Statement at the end of the main text, before references

## B. Figures

- [ ] Figures supplied as separate high-resolution files (TIFF or PDF, ≥ 300 DPI; vector formats preferred for line art).
- [ ] Each figure cited in the text in numerical order.
- [ ] Figure legends placed at the end of the main manuscript file, after the references and before any tables.
- [ ] Colour figures: ensure they remain interpretable in black-and-white print and that any colour scheme is colour-blind friendly.
- [ ] Permission obtained and credited for any reproduced figure or table.

## C. Tables

- [ ] All tables embedded in the main manuscript file in numerical order.
- [ ] Each table cited in the text in numerical order (re-number if necessary to match order of mention).
- [ ] Footnotes and abbreviation legends provided for every table.

## D. Supplementary materials

- [ ] S1 — PRISMA 2020 checklist (`supplementary/S1_prisma2020_checklist.docx`).
- [ ] S2 — Search strategies for every database, with date of execution and record counts.
- [ ] S3 — PRISMA 2020 flow data, including deduplication and cross-database overlap counts.
- [ ] S4 — Data extraction form (blank template and a worked example).
- [ ] S5 — Categorised corpus (CSV).
- [ ] S6 — Evidence-claims database (JSON).
- [ ] S7 — Full bibliography of the 340 included records.
- [ ] S8 — Citation shortlist (top records by priority score).
- [ ] S9 — Figure source files and generation scripts.
- [ ] Each supplementary file has a one-line legend in the main manuscript's Supplementary Material section.

## E. Authorship and disclosures

- [ ] CRediT author contribution statement included on the title page and reviewed by every co-author.
- [ ] Individual ICMJE Disclosure of Interest forms collected from every named author (see `/submission/icmje_coi_template.docx` for instructions).
- [ ] Manuscript Conflicts of Interest statement on the title page consistent with the aggregated content of the ICMJE forms.
- [ ] All co-authors have read and approved the final version of the manuscript and consent to submission.
- [ ] Corresponding author has confirmed correct spelling of every author's name, affiliation and ORCID identifier.

## F. Compliance and registration

- [ ] PRISMA 2020 checklist completed and supplied as S1.
- [ ] PROSPERO registration completed (CRD identifier inserted into the manuscript and title page) — or a transparent explanation provided in the methods if registration is retrospective.
- [ ] Reporting-guideline statement included in the Methods (PRISMA 2020, PRISMA-S, SWiM).
- [ ] AI-assistance disclosure included in the Data Availability Statement (or stated to be N/A).

## G. Pre-submission administrative steps

- [ ] Corresponding author has an Editorial Manager account with verified email and ORCID linked.
- [ ] Suggested reviewers (≥ 3) identified with names, affiliations and current institutional emails; none has co-authored with any author within the past three years.
- [ ] Any prior preprint posting noted in the cover letter and listed under "Prior dissemination" on the title page.
- [ ] Any prior rejection or transfer from another journal noted in the cover letter where applicable.
- [ ] Funding sources entered into Editorial Manager metadata as well as on the title page.

## H. Final checks before clicking "Submit"

- [ ] All filenames are descriptive and contain no spaces (e.g. `IJAA_QS_Abaumannii_Manuscript_v1.docx`).
- [ ] Word counts and reference numbers on the title page match the contents of the manuscript file.
- [ ] All in-text citation numbers correspond to entries in the reference list (no orphans).
- [ ] Spelling and grammar reviewed (UK or US English — choose one and apply consistently).
- [ ] One nominated author has read the final assembled submission package end-to-end in the order in which an editor will see it (cover letter → title page → manuscript → figures → tables → supplements).
